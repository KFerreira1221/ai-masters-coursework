# AI Master's Coursework Projects (FAU)

Selected graduate-level projects from the MS in Artificial Intelligence
program at Florida Atlantic University.

These projects demonstrate applied deep learning, reinforcement learning,
and large-scale graph analytics.

- Batch normalization, dropout, kernel size analysis
- TensorFlow / Keras

## CAP 6629 — Reinforcement Learning
- Gridworld environment (10×10)
- Q-learning vs SARSA comparison
- Reward shaping and convergence analysis

## Social Network Analysis — Spotify
- Large-scale artist collaboration graph
- Centrality metrics and PageRank
- PySpark modeling and clustering
